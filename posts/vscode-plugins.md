分享我日常开发中离不开的 VS Code 插件。

## 必备插件

### 1. GitHub Copilot
AI 代码补全，已经是我写代码的标配。

### 2. Prettier
统一的代码格式化，团队协作必备。

### 3. ESLint
JavaScript/TypeScript 代码质量检查。

### 4. Tailwind CSS IntelliSense
Tailwind 类名自动补全与预览。

### 5. Error Lens
将错误提示直接显示在代码行尾。

### 6. GitLens
Git 历史、Blame 信息一目了然。

### 7. Thunder Client
轻量级 API 测试工具，替代 Postman。

### 8. Markdown Preview Enhanced
实时预览 Markdown，写文档利器。

配置好工具链，让开发效率翻倍。
